# 🏥 Adimed Hospital Management System

A comprehensive, modern hospital management system built with HTML, CSS, and JavaScript, featuring a beautiful animated background and complete healthcare management capabilities.

## ✨ Features

### 🎯 Core Modules
- **👥 Patient Management** - Complete patient records with Indian localization
- **👨‍⚕️ Doctor Management** - Staff management with specializations
- **👩‍⚕️ Nursing Staff** - Nurse scheduling and department management
- **💊 Pharmacy Management** - Medicine inventory with stock tracking
- **🛏️ Bed Availability** - Real-time bed status management
- **🔬 Laboratory Services** - Test scheduling and results
- **📅 Appointment Scheduling** - Patient-doctor appointment management
- **💰 Billing System** - Indian currency billing and invoicing
- **💬 Feedback System** - Patient satisfaction tracking
- **📞 Contact Management** - Emergency and general contact information

### 🎨 Design Features
- **🌌 Animated Background** - Sophisticated multi-layer gradient animations
- **🎨 Glass Morphism** - Modern frosted glass effects
- **📱 Responsive Design** - Mobile-friendly interface
- **🇮🇳 Indian Localization** - Names, currency (₹), and contact formats
- **🎯 Professional UI** - Clean, intuitive user experience

## 🛠️ Technology Stack

### Frontend
- **HTML5** - Semantic markup structure
- **CSS3** - Advanced animations and modern styling
- **JavaScript ES6+** - Modern class-based architecture
- **Font Awesome 6.0** - Professional icon library
- **Chart.js** - Data visualization and analytics

### Features
- **Local Storage** - Data persistence without backend
- **CRUD Operations** - Create, Read, Update, Delete
- **Search & Filter** - Advanced data filtering
- **Export/Import** - CSV data management
- **Real-time Updates** - Dynamic content rendering
- **Activity Logging** - Complete audit trail
- **Form Validation** - Input validation and error handling

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional for development)

### Installation
1. **Clone or download** the project files
2. **Open `index.html`** in your web browser
3. **System ready** - All features functional immediately

### Development Setup
```bash
# Using Python (recommended)
python -m http.server 8000

# Using Node.js (alternative)
npx serve

# Using PHP (advanced)
php -S localhost
```
